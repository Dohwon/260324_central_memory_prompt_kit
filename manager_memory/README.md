# Manager Memory

이 폴더는 이 프로젝트 자체의 운영 상태를 관리한다.

## 파일 역할

- `short-term/active-tasks.md`
  - 현재 바로 실행 중이거나 곧 처리할 일
- `mid-term/current-initiatives.md`
  - 4-12주 이니셔티브
- `long-term/strategy-roadmap.md`
  - 6-12개월 전략 베팅
- `logs/YYYY-MM-DD-{task-slug}.md`
  - 날짜별 작업 로그

## 규칙

- 큰 변경을 하면 로그를 남긴다.
- 단기 상태가 바뀌면 `active-tasks.md`를 갱신한다.
- 방향 전환이나 상품화 수준의 의사결정이면 mid/long에도 반영한다.
