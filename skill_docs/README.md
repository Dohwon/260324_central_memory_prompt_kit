# Skill Docs README

이 폴더는 `skills/**/SKILL.md` 원문을 사람이 공유하기 쉬운 설명 문서로 바꾸는 계층이다.

## 왜 필요한가

- 원본 `SKILL.md`는 실행용 지침이라 길고 문맥 의존적일 수 있다.
- 공유용 설명 문서는 역할, 장점, 호출 타이밍, 협업 방법을 빠르게 보여줘야 한다.

## 현재 기준

- 원문 소스는 `skills/**/SKILL.md`
- 공유용 설명 문서는 `skill_docs/{skill-name}.md`
- 새 설명 문서는 `prompts/02_skill_profile_prompt.md`로 생성한다.
